# ML Regularized Models

A collection of experiments with machine learning algorithms focused on improving predictive performance through regularization, automated parameter search, metrics evaluation, and ensemble strategies.

## 📁 Project Structure

### regularization/
Contains a notebook demonstrating the use of regularization techniques in predictive models, including methods that reduce overfitting and help select the most informative features.

### gridsearch/
Contains a notebook exploring automated hyperparameter optimization using grid search to identify the best parameter combinations across different models.

### metrics/
Includes a notebook examining performance evaluation beyond accuracy, introducing precision, recall, F1-score, and ROC-AUC to better understand model behavior, especially for imbalanced classes.

### ensembles/
Notebook covering ensemble methods such as voting, bagging, and stacking classifiers to improve model stability and accuracy by combining predictions from multiple models.

### pipelines/
Focuses on organizing code using object-oriented programming principles and pipelines, consolidating data preparation, modeling, evaluation, and inference into reusable classes and methods.

---

## 📊 Purpose

The project demonstrates an end-to-end approach for predictive modeling:
- controlling model complexity and avoiding overfitting;
- systematically tuning hyperparameters;
- evaluating models with robust metrics;
- improving predictions using ensemble techniques;
- and structuring code for clarity and reusability.
